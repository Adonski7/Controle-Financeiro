Este projeto é um sistema web de controle financeiro, desenvolvido para registrar e gerenciar receitas e despesas de forma simples e organizada.

A aplicação permite que usuários se cadastrem, façam login e acompanhem seus lançamentos financeiros através de um painel com resumo e relatórios.

O objetivo do projeto é praticar conceitos de desenvolvimento web Full Stack, integrando frontend, backend e banco de dados.

🚀 Funcionalidades

Cadastro de usuários

Login e autenticação

Dashboard com resumo financeiro

Registro de receitas e despesas

Listagem de lançamentos

Relatórios financeiros

Logout do sistema

🛠️ Tecnologias utilizadas

Frontend

HTML5

CSS3

JavaScript

Backend

PHP

Banco de Dados

MySQL


This project is a web-based financial control system, developed to record and manage income and expenses in a simple and organized way.

The application allows users to register, log in, and track their financial transactions through a dashboard that provides summaries and reports.

The goal of this project is to practice Full Stack web development concepts, integrating frontend, backend, and database technologies.

User registration

Login and authentication

Dashboard with financial summary

Income and expense tracking

Transaction listing

Financial reports
